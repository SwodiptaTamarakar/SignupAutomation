# Signup Automation Task

# Prerequisites
- Python 3.11.1
- Playwright

# Installation
pip install -r requirements.txt
playwright install

# Run Test
pytest tests/test_signup.py -v

# Framework
- Python
- Playwright
- Pytest

1. Open website
2. Click Get Started
3. Accept agreement
4. Fill signup form
5. Handle OTP
6. Complete registration
7. Verify success
